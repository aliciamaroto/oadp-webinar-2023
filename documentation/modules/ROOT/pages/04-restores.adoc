== Restore application with persistent storage using Restic

Before we start setting up the environment, please fork the `Git` repository into your personal `Github` Account as follows:

* Access the https://github.com/aliciamaroto/oadp-webinar-2023-lab/[OADP Repository]
* Fork it on your GitHub account. If prompted, choose to fork only the `main` branch.
* Clone the repository to your local machine. You can use both SSH and HTTPS URLs. If you have already configured your SSH keys, use this mechanism. If not, use HTTPS.
