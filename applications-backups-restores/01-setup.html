Setup OpenShift Cluster

Before starting this workshop, we need to have the following environment running:

  • OpenShift 4 cluster. We will be using an OpenShift 4.13 version.

  • OpenShift Data Foundation.

  • Object Storage as Backup location. In this workshop we will use Azure Blob as external storage location.
